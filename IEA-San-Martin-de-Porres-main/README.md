# Sistema de Gestión Escolar Multisede

Una aplicación web completa y moderna para la administración integral de instituciones educativas con múltiples sedes. Diseñada para facilitar la interacción entre administradores, profesores, estudiantes y padres de familia.

## 🚀 Características Principales

*   **Gestión Multisede:** Administración centralizada para múltiples campus con configuraciones independientes.
*   **Roles y Permisos:** Sistema robusto de roles (Super Admin, Admin de Sede, Profesor, Estudiante, Padre).
*   **Gestión Académica:**
    *   Matrículas y gestión de estudiantes.
    *   Asignación de carga académica a profesores.
    *   Registro de calificaciones y asistencia en tiempo real.
    *   Generación de horarios escolares.
*   **Generador de Informes PDF:**
    *   Boletines de calificaciones.
    *   Libros finales.
    *   Consolidados y rankings.
*   **Comunicaciones:** Módulo de comunicados y mensajería interna.
*   **Inteligencia Artificial:** Integración con Google Gemini para asistencia en comentarios y generación de contenido.
*   **Interfaz Moderna:** Diseño limpio y responsivo con soporte para modo oscuro (Dark Mode).

## 🛠️ Tecnologías Utilizadas

*   **Frontend:** React, TypeScript, Tailwind CSS.
*   **Backend / Base de Datos:** Supabase (PostgreSQL, Auth).
*   **IA:** Google GenAI SDK.
*   **Utilidades:** jsPDF (Reportes), Lucide React (Iconos).

## 📋 Requisitos Previos

*   Node.js (v16 o superior)
*   Cuenta en Supabase
*   API Key de Google Gemini (Opcional, para funciones de IA)

## 📦 Instalación

1.  Clonar el repositorio.
2.  Instalar dependencias: `npm install`
3.  Configurar variables de entorno en un archivo `.env` (basado en `.env.example`).
4.  Iniciar servidor de desarrollo: `npm run dev`
